法律自助助手 - 网络部署指南
================================

【最快方式】拖拽部署（30秒获得链接）
-----------------------------------
1. 打开浏览器访问：https://app.netlify.com/drop
2. 把 C:\Users\Administrator\legal-help 整个文件夹拖入网页
3. 自动生成链接，如：https://xxx-xxx-xxx.netlify.app
4. 把这个链接发到微信即可

【永久方式】GitHub Pages（免费永久链接）
-----------------------------------
1. 注册 GitHub 账号：https://github.com/signup
2. 创建新仓库（Repository），名称任意（如 legal-help）
3. 上传 legal-help 文件夹内所有文件到仓库
4. Settings → Pages → Source: main branch → Save
5. 几分钟后链接生效：https://你的用户名.github.io/legal-help
